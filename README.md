
# 📂 Portfólio de Banco de Dados

Este repositório reúne **projetos SQL completos** para demonstração de habilidades em modelagem, criação e consulta de bancos de dados.  
O objetivo é servir como **portfólio profissional** para destacar competências na área de dados.

---

## 📌 Estrutura do Repositório
```
📂 portfolio-banco-de-dados  
 ├── 📂 Projetos  
 │     ├── 📂 sistema_vendas_db  
 │     ├── 📂 biblioteca_db  
 │     └── 📂 financeiro_db  
 ├── 📂 Scripts  
 ├── 📂 Exercícios  
 ├── 📂 Desafios  
 └── README.md
```

- **Projetos/** → Contém os bancos de dados completos, cada um com scripts e README próprio.
- **Scripts/** → Pasta para guardar scripts SQL genéricos e utilitários.
- **Exercícios/** → Atividades práticas para reforçar conceitos.
- **Desafios/** → Casos mais complexos para prática avançada.

---

## 📁 Projetos Incluídos

### 1️º Sistema de Vendas
- Gerencia **clientes**, **produtos** e **pedidos**.
- Inclui consultas para análise de faturamento.
- Scripts: criação de banco, tabelas, inserção de dados e consultas.

### 2º Sistema de Biblioteca
- Gerencia **usuários**, **livros** e **empréstimos**.
- Inclui consultas para controle de acervo e histórico de empréstimos.

### 3️º Sistema Financeiro
- Gerencia **categorias de receita/despesa** e **lançamentos**.
- Inclui consultas para saldo, total por categoria e histórico financeiro.

---

## 🚀 Como Utilizar

1. **Escolha um projeto** na pasta `Projetos/`.
2. Execute os scripts na seguinte ordem:
   1. `criar_banco.sql`
   2. `criar_tabelas.sql`
   3. `inserir_dados.sql`
   4. `consultas.sql`
3. Explore e modifique as consultas para criar novas análises.
