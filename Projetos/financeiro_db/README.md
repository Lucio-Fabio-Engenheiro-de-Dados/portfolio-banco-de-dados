💰 Sistema de Controle Financeiro - Banco de Dados

Este projeto simula um sistema de controle financeiro com categorias, receitas e despesas. Foi desenvolvido em SQL e pode ser usado como base para sistemas pessoais ou empresariais.

📌 Estrutura do Banco de Dados  

*categoria*: Tipos de movimentação (ex: Salário, Aluguel).  
*lancamento*: Entradas e saídas com data e valor.  

🚀 Como Utilizar

Execute:
Execute 'criar_banco.sql'  
Execute 'criar_tabelas.sql'  
Execute 'inserir_dados.sql'  
Teste as consultas em 'consultas.sql'  

📊 Consultas incluídas

Listagem de lançamentos  
Total de receitas, despesas e saldo  
Agrupamento por categoria  
Ordenação por data  

🛠 Tecnologias

MySQL  
GitHub Desktop  
GitHub Web  

🚀 RESUMO 

Com as consultas você aprende:

SELECT → buscar dados
WHERE → filtrar
JOIN → relacionar tabelas
GROUP BY → agrupar
SUM / AVG / COUNT → cálculos
ORDER BY → ordenar
