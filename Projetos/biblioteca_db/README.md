# 📚 Sistema de Biblioteca - Banco de Dados

Este projeto simula um sistema de gerenciamento de biblioteca utilizando SQL. Ele permite cadastrar livros, usuários e controlar empréstimos e devoluções.

## 🧩 Estrutura do Banco de Dados

- **usuario**: Armazena os leitores cadastrados.
- **livro**: Contém o acervo da biblioteca.
- **emprestimo**: Registra empréstimos e devoluções.

## 🚀 Como Utilizar

1. Execute `criar_banco.sql`
2. Execute `criar_tabelas.sql`
3. Execute `inserir_dados.sql`
4. Teste as consultas em `consultas.sql`
