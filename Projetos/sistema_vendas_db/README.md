# 📊 Sistema de Vendas — Banco de Dados
Projeto para gerenciar clientes, produtos e pedidos.
Este projeto consiste em um banco de dados SQL para gerenciar um sistema de vendas, armazenamento e informações sobre clientes, produtos e pedidos.

## Estrutura do Banco de Dados
- **cliente**: Armazena os clientes cadastrados;
- **produto**: Contém informações sobre os produtos à venda;
- **pedido**: Registra os pedidos feitos pelos clientes.

## Como utilizar
1. Execute o script - criar_banco.sql - para criar o banco de dados;
2. Execute - criar_tabelas.sql - para criar as tabelas;
3. Execute - inserir_dados.sql - para popular o banco com exemplos;
4. Use - consultas.sql - para visualizar os dados.

## Tecnologias Utilizadas
 - **Banco de Dados**: MySQL;
 - **Ferramentas**: Github, Github Desktop, Notepad++.
